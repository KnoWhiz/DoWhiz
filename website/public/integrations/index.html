Integrations

Trigger agents where work already happens.

DoWhiz supports multi-surface triggers and tool-native outcomes. Each integration follows the same model: trigger from your workspace, execute with explicit permissions, return results in context.

Last updated: February 25, 2026

Email

Trigger: Email an agent with context, files, and expected output format.

Outcome: Receive completed deliverables and next steps in-thread.

Slack

Trigger: Message an agent or @mention in the relevant channel.

Outcome: In-channel updates, structured task output, and follow-up prompts.

Discord

Trigger: DM or channel mention to route requests to the right agent.

Outcome: Summaries, action plans, and linked deliverables returned to Discord.

GitHub

Trigger: Create or assign an issue to an agent workflow.

Outcome: Implementation updates, PRs, tests, and concise technical summaries.

Google Docs

Trigger: @mention an agent in comments on a shared document.

Outcome: Action item extraction, rewrites, summaries, and status-ready edits.

Notion

Trigger: Mention agents in shared pages/tasks and attach workspace context.

Outcome: Updated plans, task breakdowns, and structured content in-place.
